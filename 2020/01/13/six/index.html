小程序—点击收藏功能(刷新状态依然存在)

最近在做小程序中商品的收藏功能，网上也能找到很多解决方法，但是收藏成功后，重新加载页面，数据仍然保存在localstorage中，但是收藏的样式恢复到了初始值。

wxml中的代码如下：

<view bind:tap="toCollect" class="iconfont {{isCollect?'icon-collection':'icon-shoucang'}} "> </view> <view>收藏</view>

收藏样式切换

开始之前，我的思路是通过页面加载来获取缓存中的状态，这里先做一个判断如果缓存中没有值，加载时向缓存中添加状态false，（当然你不添加也可以，因为没点击收藏之前此状态就是false，但是要明白原理！），如果有值，获取缓存中的值，点击时取反，在渲染到页面。

js代码：

1.在data中设置iscollect

isCollect: false,

2.获取缓存状态方法和点击收藏按钮显示收藏成功

toCollect: function () {
let isCollect = false
let collect = wx.getStorageSync('collect') || [];

let index = collect.findIndex(v => v.id === this.goodsInfo.id);
if (index !== -1) {
  // 收藏过
  collect.splice(index, 1);
  isCollect = false
  wx.showToast({
    title: '取消收藏',
    icon: 'success',
    mask: true
  })
} else {
  // 没收藏过
  collect.push(this.goodsInfo);
  isCollect = true
  wx.showToast({
    title: '收藏成功',
    icon: 'success',
    mask: true
  })
}
// 缓存
wx.setStorageSync("collect", collect);
// 修改data 中的数据
this.setData({
  isCollect
});
console.log(index)

},

getproductdetail: async function () {

let data = await getRequest("https://www.zixibeng.shop/addons/zjhj_mall/core/web/index.php?_acid=69&r=api/default/goods",
  { id: this.id })

//goodsInfo是放收藏的数据
this.goodsInfo = data.data.data

let collect = wx.getStorageSync('collect') || [];
let isCollect = collect.some(v => v.id === this.goodsInfo.id)

this.setData({

  isCollect
